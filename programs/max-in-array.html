Find Max in Array

This program finds the maximum number in an integer array.
It iterates through the array and keeps track of the largest value.

Java Code 
// Java program to find maximum element in an array
public class FindMaxInArray {
    public static void main(String[] args) {
        int[] arr = {3, 7, 2, 9, 5};

        int max = arr[0]; // Assume first element is max initially

        // Loop through array to find max
        for (int i = 1; i < arr.length; i++) {
            if (arr[i] > max) {
                max = arr[i];
            }
        }

        System.out.println("Maximum element is: " + max);
    }
}
Output
Maximum element is: 9
